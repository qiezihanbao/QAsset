TAMAGO - 产品需求规格说明书 (PRD)
1. 项目背景与总体目标
1.1 业务背景
随着数字创意产业的发展，设计团队与个人创作者面临着资产多元化、跨地域分布及数量激增的挑战。根据行业调研，超过72%的企业在资产管理数字化过程中因需求不清导致返工 。传统的文件系统无法满足对图片、视频、3D模型等非结构化数据的深度检索（如颜色、形状）与全生命周期管理需求。
1.2 项目目标
本项目旨在构建一款基于 Rust + Tauri v2 的跨平台美术资产管理软件，覆盖 PC、Android 及 Web 端。系统旨在实现：

资产透明化：实现资产从导入、整理、应用到归档的全生命周期管理 。
检索智能化：支持基于颜色占比、标签、元数据的多维检索。
协作高效化：通过 NAS 同步与工作区机制，解决团队协作与跨设备办公痛点。

2. 用户角色与业务流程
2.1 用户角色定义

个人创作者：关注素材的快速收集、灵感整理与跨设备查看

2.2 核心业务流程
系统需支持资产的全生命周期流转，确保流程合规与数据留痕 ：

资产登记：支持拖拽导入、浏览器插件采集，自动提取元数据（分辨率、颜色信息等）。
整理入库：用户打标签、分类、评分；系统自动进行重复检测与丢失文件扫描。
应用流转：用户在工作区调用资产，或通过筛选器查找特定素材。
同步协作：数据加密同步至 NAS 或云端，支持多端访问。
归档/报废：支持资产的逻辑删除（进入暂存区）与物理删除，并记录操作日志 。

3. 功能模块详细设计
系统划分为八大核心功能模块，确保功能的完整性与可扩展性 。
3.1 资源库管理模块

多库支持：支持创建多个独立资源库，彼此物理隔离(因为要上传NAS管理 所以实现麻烦也可以是单库)

健康度监控：

丢失文件检测：启动时自动扫描，标记丢失文件并提供“自动搜索重连”与“手动重连”功能。
版本控制：记录资产的关键修改历史，支持回滚。

3.2 资产台账与存储模块

台账登记：自动记录资产名称、来源、大小、格式、创建时间等基础信息 。
智能分类：

物理目录：同步显示文件系统目录结构。
逻辑分类：支持多级标签组、智能文件夹（基于筛选条件自动更新）。

格式支持：原生支持图片、视频、音频、3D模型及文档的缩略图生成与预览。

3.3 智能检索与筛选模块

多维筛选：支持按名称、标签、类型、日期、评分进行组合查询 。
视觉检索：

颜色占比筛选：基于 K-Means/HSV 算法，支持按颜色占比阈值筛选图片。
相似图查找：基于感知哈希算法查找视觉相似资产。

高级搜索：支持布尔逻辑（AND/OR）查询，并可保存搜索预设。

3.4 工作区模块

虚拟空间：用户可创建多个临时工作区，跨文件夹引用资产，不移动物理文件。
状态持久化：自动保存工作区内的布局、排序与备注，支持断点续传。

3.5 同步与协作模块

NAS/云端同步：支持 WebDAV/SMB 协议，实现多端数据同步。
端到端加密：传输与存储过程全程加密，保障数据安全 。
冲突处理：自动处理多端修改冲突，遵循“最新修改优先”或“人工确认”策略。

3.6 系统管理模块

权限控制：支持角色与权限分级管理，实现最小授权原则 。
操作日志：记录所有关键操作（删除、修改、导出），支持审计追溯 。

4. 用户界面设计规范
界面设计应遵循“简洁、高效、自适应”原则，确保跨端体验一致性 。
4.1 布局架构
采用经典的三栏式布局，支持自定义显隐：

顶部导航栏：全局搜索、资源库切换、布局视图切换（网格/瀑布流/列表/自适应）。
左侧边栏：

上部：逻辑视图（全部文件、待整理、标签管理、回收站、工作区）。
下部：物理文件夹树状图。

右侧详情栏：展示选中资产的元数据、标签编辑、预览及“丢失文件”修复入口。

4.2 交互体验

响应式设计：适配 PC 大屏与移动端竖屏布局。
操作便捷性：支持拖拽导入、右键菜单快捷操作、键盘快捷键。

5. 非功能性需求
5.1 性能要求

响应速度：万级资产库的常规筛选响应时间应低于 500ms 。
文件最好不要像eagle打破原来的文件夹布局 基于原生文件夹结构 磁盘IO友好
并发处理：支持后台异步生成缩略图与提取颜色特征，不阻塞 UI 主线程。

5.2 安全性要求

数据安全：利用 Rust 语言特性防止内存溢出漏洞；敏感数据（密码、密钥）加密存储。


5.3 可扩展性

插件架构：采用 Tauri 插件系统，便于后期扩展新的文件格式解析（如 USDZ）或 AI 分析功能。
API 开放：预留 API 接口，支持与 ERP、OA 等第三方系统对接 。

6. 开发路线图

第一阶段（MVP）：完成基础架构搭建、文件导入、元数据提取、基础筛选及丢失文件检测。
第二阶段（进阶）：实现颜色占比筛选、工作区功能、NAS 同步。
第三阶段（生态）：开放 API 接口，推出浏览器插件，引入 AI 自动标签功能。
